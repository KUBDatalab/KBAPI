## Installation

You can install the development version of KBapi like so:

``` r
options(repos = c(
  kubdatalab = '',
  CRAN = 'https://cloud.r-project.org')
)

install.packages('KBAPI')
```

Or directly from github:

``` r
devtools::install_github('KUBDatalab/KBAPI')
```
